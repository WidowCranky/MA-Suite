# MA-Suite

Custom indicators for Quantower.

## Indicators

### SmaEmaCross
Plots a Simple Moving Average and an Exponential Moving Average.  
**New:** Optional crossover arrows (green up / red down) when the faster line crosses the slower one.

### DualHma
Plots two Hull Moving Averages with configurable periods and colors.

### TripleSma
Plots three Simple Moving Averages.

### TripleEmaHlc
Three EMAs (Close, High, Low) + optional Wilder's SMMA smoothing line.

## Installation

1. Download or build the `MA_Suite.dll`
2. Copy it to your Quantower folder:  
   `Settings\Scripts\Indicators\MA_Suite\`
3. Restart Quantower or refresh indicators.
